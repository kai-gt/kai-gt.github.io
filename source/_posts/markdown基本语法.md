## 标题(h1 - h6)

```markdown
# h1
## h2
### h3
#### h4
##### h5
###### h6
```

---

## 段落及区块引用( > )

```markdown
> 这段内容将被高亮...
```

> 这段内容将被高亮...

---

## 插入链接或图片

```markdown
    [点我去百度](http://www.baidu.com)
    ![头像](https://avatars.githubusercontent.com/u/68783565?v=4)
```

> [点我去百度](http://www.baidu.com)
> ![头像](https://avatars.githubusercontent.com/u/68783565?v=4)

---

## 列表 无序:(*、+、-) 有序:(1.)

```markdown
1. 无序：
   * 小明
   + 小红
   - 小白

2. 有序:
   1. 小明
   2. 小红
   3. 小白
```

1. 无序
   * 小明
   + 小红
   - 小白
2. 有序
   1. 小明
   2. 小红
   3. 小白

---

### 分割线

```markdown
---
上下两个都为分割线
***
```

---
上下两个都为分割线
***

### 强调 (加粗或者斜体)

```markdown
*斜体*
_斜体_

**加粗**
__加粗__
```

> *斜体*
> _斜体_
> **加粗**
> __加粗__

---

### 代码块

```markdown
   \```(语言)
    console.log('hello word!')  \ 为转义
   \```
```

```js
console.log('hello word!')
```

### 插入表格

> 两端均有冒号(居中) 左边有(居左) 右边有(居右)

```markdown
表头一|表头二|表头三
:---|---:|:---:
项目1|项目2|项目3
```

表头一|表头二|表头三
:---|---:|:---:
项目1|项目2|项目3
